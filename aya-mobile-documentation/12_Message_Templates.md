## 🔗 WhatsApp Deep Links

### كيفية بناء الرابط

```javascript
const encodeMessage = (text) => encodeURIComponent(text);
const phoneNumber = "962XXXXXXXXX"; // رقم بدون + أو 00
const message = "نص الرسالة";
const link = `https://wa.me/${phoneNumber}?text=${encodeMessage(message)}`;
```

### ملاحظات

- الرقم يجب أن يكون بصيغة دولية (962 للأردن)
- لا تضع + أو 00 قبل الرقم
- الرسالة يجب أن تكون URL encoded
- يفتح واتساب مع الرسالة جاهزة للإرسال
- النظام يسجل `delivery log` محليًا عند تشغيل زر واتساب، لكن هذا السجل يثبت intent/audit فقط ولا يساوي confirmation من مزود خارجي

## u{1F514} تدفق إرسال واتساب من الإشعارات

`
u{1F514} 3 إشعارات → [قائمة الإشعارات]
  u{1F534} دين متأخر  — أبو محمد  250 د.أ  متأخر 5 أيام      [u{1F4F1} واتساب]
  u{1F7E1} دين قريب   — خالد صالح 100 د.أ  يستحق بعد يومين   [u{1F4F1} واتساب]

↓ يضغط [u{1F4F1} واتساب]

[Modal: اختر الرسالة]
  العميل: أبو محمد  | الهاتف: 079-xxx-xxxx
  u{25CB} تذكير ودي  u{25CF} تذكير رسمي
  [معاينة الرسالة — قابلة للتعديل]
  [تعديل]  [فتح واتساب u{2197}]

↓ يضغط [فتح واتساب] → Chrome يفتح wa.me/962... → أحمد يضغط إرسال
`

### متى يظهر زر واتساب في الإشعار؟

| نوع الإشعار | يظهر الزر؟ | الشرط |
|------------|-----------|-------|
| `debt_overdue` | u{2705} دائماً | دين متأخر |
| `debt_due_reminder` | u{2705} دائماً | يستحق ≤ 3 أيام |
| `debt_limit_exceeded` | u{2705} إذا يوجد هاتف | phone غير فارغ |
| باقي الإشعارات | u{274C} | لا واتساب |

**إذا الهاتف فارغ:** يظهر زر **نسخ الرسالة** فقط.
